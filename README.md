1. **会记录我的聊天内容吗？**  
   部分站点为提升服务质量会匿名存储对话，但支持在设置中关闭此功能。  

2. **GPT-4和GPT-3.5如何选择？**

   - GPT-3.5：适合日常对话、简单文案（响应更快）  

3. **遇到连接超时怎么办？**  
   尝试切换节点（镜像站通常提供CN/HK/US线路）或清理浏览器缓存。
